# Changelog

All notable changes to this project are documented here.

## [Unreleased]

## [1.0.1] - 2026-07-10

### Added
- Install helper: opening the app with `#install` (from apps.riverma.com) shows an install prompt or Add-to-Home-Screen guidance, so the launcher's Install button installs this app correctly.

## [1.0.0] - 2026-07-10

### Added
- First independent release of Mobility Choices, extracted from `riverma.github.io` into its own repository and published at [mobility.riverma.com](https://mobility.riverma.com).
- Self-hosted all third-party libraries (removed CDNs); only live map/route data is fetched at runtime.
- In-app footer with a privacy statement, version, and source/license link.
- AGPL-3.0 license.
